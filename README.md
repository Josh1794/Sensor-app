## Description

An app that allows users to explore the various sensors built into a modern smartphone

## Features

- Accelerometer
- Barometer
- Gyroscope
- Location (Latitude, Longitude, Heading)
- Magnetometer
- Pedometer
- Dark mode/Light mode based on system settings
